## ✨ Features

### 🎯 Core
- Floating Overlay Bubble — Stays on top of every app
- Live Speech Recognition — Continuous listening
- Smart Speaker ID — Interviewer vs user detection
- Real-time AI Responses — Streaming answers
- Transparent Chat Window — Overlay UI
- Hold to Move — Long press to drag

### 🧠 AI Intelligence
- Question Detection
- IT Domain Keywords
- Speech Correction
- Context Memory
- Interview Mode

### 🔌 Multiple AI Providers

| Provider | Models | Cost |
|----------|--------|------|
| NVIDIA NIM | Llama, Mistral | Free |
| Gemini | Flash/Pro | Free tier |

## 🚀 Quick Start

### Requirements
- Android 8+
- API Key (NVIDIA or Gemini)
- Microphone + Overlay permission

### Steps
1. Install APK
2. Add API key
3. Start overlay
4. Speak → AI responds
